# AI-Based Diabetes Risk Prediction System

This project applies machine learning and explainable AI techniques to predict diabetes risk using clinical health indicators.

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- SHAP Explainability
- Power BI

## Project Pipeline

1. Data Cleaning
2. Exploratory Data Analysis
3. Machine Learning Models
4. Model Evaluation (Accuracy, Precision, Recall, ROC-AUC)
5. Feature Importance
6. SHAP Explainability
7. Power BI Dashboard

## Key Insight

SHAP analysis revealed that glucose level, BMI, and age are the strongest predictors of diabetes risk.

## Dashboard

Power BI dashboard visualizes:

- Diabetes risk distribution
- Feature importance
- Prediction accuracy
- Model confidence
