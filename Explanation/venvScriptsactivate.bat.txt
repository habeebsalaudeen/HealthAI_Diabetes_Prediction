🔥 NEXT STEP: Identify invalid zeros (VERY IMPORTANT IN MEDICAL DATA)

PERFECT interview answer (save this)

“I only treated zeros as missing in variables where a value of zero is physiologically impossible, such as glucose, blood pressure, BMI, insulin, and skin thickness. For variables where zero is meaningful, like pregnancies or the outcome label, I left them unchanged.”

CELL — Check invalid zero values

1️⃣ Glucose

Plasma glucose concentration

A value of 0 → patient would be dead

So 0 = missing measurement

2️⃣ BloodPressure

Diastolic blood pressure

0 mm Hg → no blood circulation

Impossible in real life

3️⃣ SkinThickness

Triceps skinfold thickness

0 mm → measurement not taken

4️⃣ Insulin

Serum insulin level

0 → lab result missing or not recorded

5️⃣ BMI

Body Mass Index

BMI = 0 → impossible for a human body

👉 These zeros are data-collection artifacts, not real values.

✅ Columns where 0 IS VALID (do NOT touch)
✔ Pregnancies

A woman can have 0 pregnancies

0 is a real, meaningful value

✔ Age

Age is never 0 in this dataset, but if it were, it would be obvious

No need for zero handling here

✔ DiabetesPedigreeFunction

A continuous genetic risk score

Zero is mathematically possible

✔ Outcome

Target variable

0 = non-diabetic

1 = diabetic

This is by definition, not an error
